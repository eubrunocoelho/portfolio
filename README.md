## Instalação

### Requisitos

-   [Node.js](https://nodejs.org/pt)
-   [NPM](https://www.npmjs.com/)

```
# Clonando o projeto
git clone https://github.com/eubrunocoelho/portfolio.git

# Instalando as dependências
npm install

# Executando o live server
npm run serve
```
